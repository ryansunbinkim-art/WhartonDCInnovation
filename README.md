# Wharton DC Innovation Summit

Production-ready static website built from the [Figma Make conference design](https://www.figma.com/make/qJaUNCK28O9GNhSpkDtdtG/Conference-website-outline).

## Run locally

```bash
cd ~/Projects/wharton-innovation-summit
python3 -m http.server 8080
```

Open http://localhost:8080

## Features

- Hero parallax zoom, staged intro text, scroll indicator
- Smooth nav scroll with fixed header offset
- Scroll-reveal sections (IntersectionObserver + CSS blur/fade)
- Animated stat counters
- Auto-scrolling sponsor logo carousel (pause on hover, grayscale to color)
- Full agenda, breakout tracks, speakers, sponsors, reviews, pricing
- YouTube highlights embed
- Responsive mobile nav drawer

No build step required — plain HTML, CSS, and JavaScript.
